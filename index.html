LASU Certificate Generator

Usage Guide — Step-by-step instructions for building, running, and using the application

Guidance on usage: lasu-csc226-group6.vercel.app Video tutorial: How to use

Build Run Use Generate

1 Build the Application

Open a terminal in the project directory and compile using g++ with C++17 standard and the Winsock2 library:

g++ -std=c++17 -I include -o certificate_generator.exe main.cpp src/*.cpp -lws2_32

This produces certificate_generator.exe in the current directory. The -I include flag tells the compiler where to find header files, and -lws2_32 links the Windows Socket library.

ⓘ

Alternative: If you have CMake installed, run cmake -B build && cmake --build build to use the provided CMakeLists.txt.

2 Run the Server

Launch the application from the terminal:

./certificate_generator

You should see output like this:

  ========================================
   LASU Certificate Generator v2.0
   Lagos State University
   Department of Computer Science
   CSC 226 — Group 6
  ========================================

  ========================================
   Certificate Generator Web UI
  ========================================
   Server running at: http://localhost:8080
   Opening in your browser...
  ========================================
   Press Ctrl+C to stop.

The application will automatically open your default browser to http://localhost:8080. If it doesn't, manually open that URL.

Terminal showing the server starting on http://localhost:8080

3 Dashboard Overview

When you first open the browser, you'll see the Dashboard. It shows four stat cards:

  • Total Students — how many students are currently loaded
  • Average GPA — the mean GPA across all students
  • Certificates Generated — count of certificate files created
  • Active Courses — number of unique courses

The dashboard also has quick action buttons: Add Student, Generate Certificates, and Import CSV.

Dashboard with stat cards — Total Students, Average GPA, Certificates, and Courses

4 Load Sample Data from CSV

The project includes a sample file students.csv with 10 student records. To load it:

  1. Click the Students tab
  2. Click the Import from CSV button — a file picker will open
  3. Select students.csv from the project folder and click Open
  4. You'll see a success notification and the table will populate

You can also click the ? button next to Import to see the required CSV format.

Students tab showing the populated table with grade badges

5 Add a Student Manually

You can add students one at a time using the add form in the Students tab:

  1. Enter the student's name (e.g., "Test Student")
  2. Enter the course (e.g., "CSC 226")
  3. Enter a valid grade — letter grades (A, B+, A-) or numeric (0-100)
  4. Click Add Student

Invalid grades (e.g., "Z" or "-5") will be rejected with a validation error notification.

Add Student form with name, course, and grade fields filled in

6 Search and Sort Students

Use the Search field in the Students tab to find specific students by name or course. The search is case-insensitive and matches partial text.

Click the Remove button next to any student to delete them from the database. Sorting is available by name, grade (ascending/descending), and course.

Search results filtered by course name — case-insensitive partial matching

7 Generate Certificates

Go to the Generate tab. You'll see three dropdowns:

Certificate Type
  • Excellence — for outstanding academic achievement
  • Completion — for successful course completion
  • Participation — for active participation
Template Style
  • Formal — serif font, double border, traditional
  • Modern — gradient background, rounded corners
  • Minimalist — monospace font, clean & simple

To generate:

  1. Select a Certificate Type and Template Style
  2. Leave the Student Name field empty to generate for ALL students, or enter a specific name
  3. Click Generate Certificate or Generate for All Students
  4. Generated files appear in the list below — click View to open in a new tab
Generate tab with certificate type and template style selected, certificates listed below

8 View Certificate Output

Each generated certificate is an HTML file saved to the output/ directory. The filename format is StudentName_CertificateType_Style.html (spaces replaced with underscores).

Click any filename in the certificates list to open it in a new browser tab. From there, press Ctrl+P to print it as a physical certificate.

Formal Template — serif font, double border, traditional design
Modern Template — gradient background, rounded corners
Minimalist Template — monospace font, clean simple design

9 View Statistics

The Statistics tab provides two breakdowns:

  • Course Distribution — how many students are enrolled in each course
  • Grade Distribution — how many students achieved each grade

Statistics update automatically whenever you add or remove students. The top stat cards show the total student count and average GPA.

Statistics tab showing course distribution and grade distribution

10 Theme Toggle

Click the sun/moon/monitor icons in the top-right corner to switch between Light, Dark, and System themes. Your preference is saved in the browser's local storage.

Light mode theme — click the sun icon in the header

Stopping the Server

When you're done, go back to the terminal and press Ctrl+C. The server will shut down gracefully.

Troubleshooting

Port 8080 already in use? Change the port in main.cpp:49 from 8080 to another number (e.g., 3000) and rebuild.

Compiler not found? Install MinGW-w64 for Windows from winlibs.com. Ensure g++ is in your PATH.

"Winsock not found"? Make sure you're on Windows and have the Windows SDK installed. This application only runs on Windows.

No output directory? The application creates output/ automatically when generating certificates.
